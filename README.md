# Installation Instructions
- Download the rbxmx file that's in the repository
- Press Win + R on your keyboard and **Run** should open up
- Type %localappdata% and press Enter, this should open up your local app data folder
- Find Roblox then open up the Plugins folder
- Move the rbxmx file you just downloaded in to your Plugins folder
- Launch Studio and if you see the plugin button, you're done!
